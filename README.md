# Sage Edu. Tech – An AI-Powered Learning Platform 🎓

## Intelligent. Interactive. Infinite Possibilities.

---

### 🌍 Overview
Sage Edu. Tech is a cutting-edge educational platform that harnesses the power of artificial intelligence to redefine how students learn and grow. Launched in **October 2023**, this **full-stack application** integrates **interactive tools, personalized recommendations, and predictive analytics** to create a dynamic, engaging, and tailored learning experience.

Built with a modern tech stack—**React.js, Node.js, Python, and MongoDB**—Sage Edu. Tech combines a sleek, user-friendly frontend with a robust, scalable backend. From **AI-driven tutors** to **gamified quizzes** and **multimedia-rich content**, this platform empowers learners while providing educators with actionable insights through **Power BI dashboards**.

## ✨ Features
✅ **AI-Powered Learning** – Personalized course recommendations and performance predictions via scalable ML models.  
✅ **Interactive Tools** – Dynamic quizzes, gamification elements, and multimedia content for an immersive experience.

✅ **Scalable Architecture** – Designed to handle growing datasets and user bases effortlessly.

## 🚀 Getting Started

### 📌 Prerequisites
Ensure you have the following installed:
- **Node.js** (v16.x or later)
- **Python** (v3.9 or later)
- **MongoDB** (local or Atlas)
- **Power BI Desktop** (for analytics dashboard)

### 📂 Installation
#### 1️⃣ Clone the Repository
```bash
git clone https://github.com/rishirai13/Sage-Edu.-Tech.git
cd Sage-Edu.-Tech
```
#### 2️⃣ Install Frontend Dependencies
```bash
cd frontend
npm install
```
#### 3️⃣ Install Backend Dependencies
```bash
cd ../backend
npm install
pip install -r requirements.txt
```
#### 4️⃣ Configure Environment
Create a `.env` file in the backend directory:
```env
MONGO_URI=your_mongodb_uri
NODE_PORT=5000
PYTHON_API_PORT=8000
```
#### 5️⃣ Run the Application
**Frontend:**
```bash
cd frontend
npm start
```
**Backend (Node.js):**
```bash
cd backend
npm run server
```
**Backend (Python ML API):**
```bash
cd backend
python api.py
```

📍 Open **http://localhost:3000** in your browser to start learning!

## 🌍 Deployment
To deploy Sage Edu. Tech:
1. Host the **frontend** on a service like **Vercel or Netlify**.
2. Deploy the **Node.js backend** and **Python ML API** on **Heroku, AWS, or similar platforms**.
3. Set up **MongoDB Atlas** for cloud-based database management.
4. Connect **Power BI** to your MongoDB instance for real-time analytics.
